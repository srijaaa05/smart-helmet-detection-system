# Smart Helmet Detection System

## Overview

SafetyAI is a computer vision-based safety monitoring system designed to detect helmets in real-time video streams and surveillance footage. The project leverages YOLOv3 and OpenCV to identify safety compliance violations and assist organizations in improving workplace safety standards.

## Features

* Real-time helmet detection using webcam or video input
* YOLOv3-based object detection architecture
* OpenCV-powered image and video processing
* Automated safety compliance monitoring
* Bounding box visualization for detected objects
* Live video stream analysis

## Technologies Used

* Python
* OpenCV
* YOLOv3
* NumPy
* Computer Vision

## Project Highlights

* Developed a real-time safety monitoring solution for helmet compliance detection.
* Implemented deep learning-based object detection using YOLOv3.
* Enabled automated monitoring through live video streams and recorded footage.
* Designed for deployment in industrial, construction, and campus environments.

## Applications

* Construction Site Monitoring
* Industrial Safety Compliance
* Smart Surveillance Systems
* Campus Safety Monitoring

## Output

The system detects helmet usage in real time and highlights detected objects with confidence scores, helping organizations identify safety violations efficiently.
